# 《深入理解Java虚拟机》学习笔记

## 工具
- JConsole：Java监视与管理控制台  
  JDK自带，路径：`%JAVA_HOME%\bin\jconsole`
- VisualVM(All-in-One Java Troubleshooting Tool)  
  JDK自带，路径：`%JAVA_HOME%\bin\jvisualvm`。网址：https://visualvm.github.io/index.html
